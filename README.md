
# TODO

In our daily lives, we often forget important tasks or rely on notes and papers, which are not always convenient to carry. Since we spend a significant amount of time on our laptops, this project provides a simple and efficient solution.

This TODO List application allows users to manage their daily tasks directly on their local system. Users can easily add tasks they plan to complete and remove them once finished. The application is designed with a clean and user-friendly interface, making task management simple and effective.

## Features

- Add daily tasks easily
- Remove completed tasks
- Simple and clean user interface
- Runs locally on your PC Browser
